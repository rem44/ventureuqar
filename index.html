MacVision AI

Système de détection anomalique pour Machines à Coudre Automatisées

Contexte

Les lignes de production de carpettes touffetées sont équipées de Machines à Coudre Automatisées (MAC) qui nécessitent une surveillance constante pour maintenir une qualité optimale. Actuellement, chacune des 3 MAC est équipée de 8 caméras qui ne sont pas pleinement exploitées pour la détection proactive de défauts.

Objectif

Développer un système de détection d'anomalies exploitant les caméras existantes pour identifier en temps réel :

  • Défauts de maintenance : désalignement des têtes, bouches d'aspiration bouchées
  • Défauts de qualité : mauvaise cautérisation, défauts sur l'endos et la face

Compétences Développées

Intelligence artificielle Vision par ordinateur C# Windows Forms Apprentissage non supervisé Traitement d'images Systèmes temps réel

Phases du Projet (12 semaines)

2 sem
Analyse et configuration

Étude du système existant, environnement de développement

2 sem
Acquisition des données

Module de capture, collection d'échantillons, prétraitement

3 sem
Développement du modèle

Autoencodeurs, One-class SVM, filtres spécialisés

2 sem
Interface utilisateur

Windows Forms C#, système de notification

2 sem
Intégration et tests

Tests en production, optimisation

1 sem
Finalisation

Formation, documentation technique

Livrables

  • Application Windows Forms en C# pour la détection de défauts
  • Modèles d'IA entraînés pour chaque type de défaut
  • Système d'alerte en temps réel
  • Documentation technique complète
  • Code source documenté et structuré

PrésenGen

Assistant IA de création de présentations commerciales personnalisées

Contexte

Les représentants commerciaux consacrent un temps considérable à préparer des présentations personnalisées pour les clients potentiels, ce qui réduit leur disponibilité pour les interactions directes avec les clients.

Objectif

Développer un système intelligent capable de générer automatiquement des présentations commerciales personnalisées en fonction du profil client, incluant les designs de tapis appropriés, les formats disponibles, et adaptant l'argumentaire aux objections typiques du secteur du client.

Compétences Développées

IA générative Traitement de données UI/UX Design Automatisation Bases de données Intégration bureautique

Phases du Projet (16 semaines)

3 sem
Analyse des besoins

Entretiens avec les représentants, analyse des présentations

4 sem
Base de données

Structuration des données produits, classification clients

4 sem
Moteur de génération

Algorithme de sélection, système de génération

3 sem
Interface et tests

Interface intuitive, tests avec cas réels

2 sem
Déploiement

Optimisation, formation des représentants

Livrables

  • Application fonctionnelle de génération de présentations
  • Base de données structurée de produits et arguments commerciaux
  • Guide utilisateur complet
  • Documentation technique
  • Formation des utilisateurs finaux
